# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-14

### Added
- Initial release of react-native-toast-message-ts
- Full TypeScript support with comprehensive type definitions
- Four pre-styled toast variants: success, error, warning, info
- Custom toast type with full customization options
- Smooth animations with native driver support
- Swipe to dismiss functionality
- Position options: top and bottom
- Customizable colors, icons, durations, and styles
- Callback support: onPress, onShow, onHide
- Global configuration through ToastContainer
- Custom toast renderer support
- Comprehensive test suite with 80%+ coverage
- Full documentation with usage examples
- Cross-platform support (iOS and Android)
- Zero external dependencies
- Persistent toast support (duration: 0)
- Auto-hide functionality with configurable duration
- Custom icon support
- Event-based architecture for flexibility
