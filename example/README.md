# Example App

This is an example app demonstrating all features of react-native-toast-message-ts.

## Running the Example

### Install dependencies

```bash
cd example
npm install
```

### iOS

```bash
cd ios
pod install
cd ..
npm run ios
```

### Android

```bash
npm run android
```

## Features Demonstrated

- Basic toast variants (success, error, warning, info)
- Different positions (top, bottom)
- Custom durations
- Persistent toasts
- Custom styling (colors, fonts)
- Interactive toasts (onPress callbacks)
- Swipeable and non-swipeable toasts
- Fully configured custom toasts
